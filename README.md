## 如何运行这个项目

现阶段只运行在苹果设备中，android 设备这边还未测试，问题应该也不大

```sh
fvm install
fvm flutter run ios 
```

通过 fvm 控制项目的flutter 版本。如果您还未使用过，可以参考其[官方文档](https://fvm.app/docs/getting_started/installation)
